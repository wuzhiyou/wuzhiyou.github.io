两种接收数据的格式—json和xml

2024-05-13 日常随笔 230 字
web

前言：用语雀复习分布式的时候对code块的xml和json格式有点疑问。

json实例：

{
    "sites": [
    { "name":"菜鸟教程" , "url":"www.runoob.com" }, 
    { "name":"google" , "url":"www.google.com" }, 
    { "name":"微博" , "url":"www.weibo.com" }
    ]
}

xml实例：

<sites>
  <site>
    <name>菜鸟教程</name> <url>www.runoob.com</url>
  </site>
  <site>
    <name>google</name> <url>www.google.com</url>
  </site>
  <site>
    <name>微博</name> <url>www.weibo.com</url>
  </site>
</sites>

SON 与 XML 的不同之处：

  • JSON 不需要结束标签
  • JSON 更加简短
  • JSON 读写速度更快
  • JSON 可以使用数组

最大的不同是：

XML 需要使用 XML 解析器来解析。

JSON 可以使用标准的 JavaScript 函数来解析。
